# Representation graphique de fonctions.

> Programme python qui trace une courbe qui suit les point de quatre fonctions sur un graphique crée avec la bibliothèque Turtle 
---
## Fonctionnement
> Pour changer les fonctions que le programme execute, il faur aller aux lignes **61**, **68**, **75**, **82** et ecrire les fonctions dans les differentes variables **x**.
---
#### Installation.
* pip install turtle
